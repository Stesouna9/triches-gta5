# Triches GTA V — Mode histoire

Référence mobile-first de tous les codes de triche de GTA V (mode solo).
Recherche live, onglets par méthode (Téléphone / PlayStation / Xbox / PC),
combos manette en pastilles de boutons, copie au tap.

**En ligne :** https://stesouna9.github.io/triches-gta5/

## Déploiement
Fichier unique autonome. Aucune dépendance, aucun build.
Pousse `index.html` sur une branche `main`, puis active GitHub Pages
(Settings → Pages → Source : `main` / `(root)`).
